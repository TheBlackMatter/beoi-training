# Unit 25: Game theory
This unit covers the basics of game theory and some classical algorithms:

- Basics and game trees
- Bachet's game
- Nim game
- Minimax algorithm
- Alpha-beta pruning

## Prerequisites
- Unit 1: [Complexity](../01-complexity)
- Unit 13: [Dynamic Programming I](../13-dynamic-programming-i)
